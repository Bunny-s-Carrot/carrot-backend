### 2. Quick Features  

- 사용자 인증: 회원가입 / 로그인`(JWT 인증)` / 로그아웃 / 탈퇴
- 이미지 버켓: BackBlaze
- 서버 배포: Cloudtype
- Cloud DB: AWS RDS / MongoDB Atlas
